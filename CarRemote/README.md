# Car Remote

[Waveshare ESP32-S3 Touch AMOLED 1.8 inch](https://www.waveshare.com/wiki/ESP32-S3-Touch-AMOLED-1.8)

- Serves as the car beacon to allow distance estimation
- Displays the current garage state and signal strength
- Controls the garage via the touchscreen

## Libraries

- Arduino_GFX (display)
- Adafruit_XCA9554 (touch)
- Arduino_DriveBus (touch)
