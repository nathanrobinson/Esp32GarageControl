# Garage Sensor

[Waveshare ESP32-S3 Touch LCD 4.3 inch](https://www.waveshare.com/wiki/ESP32-S3-Touch-LCD-4.3)

- Serves as the sensor that does distance estimation
- Displays the current garage state and signal strength
- Controls the garage via the touchscreen

## Libraries

- Arduino_GFX (display)
- TAMC_GT911 (touch)
- PubSubClient (mqtt)
